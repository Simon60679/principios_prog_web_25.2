Principios Prog. Web

Miniteste 1

Um projeto de marketplace simples, com funcionalidades tradicionais.

Principais funcionalidades

  • Busca de produto(s) (detalhamento, listagem)
  • Gerenciamento de pedidos/vendas (histórico, status, etc)
  • Gestão de catálogo e estoque

Perguntas

  • Pergunta 1

    • Mecanismo de busca de produtos

    Gerenciamento de pedidos e/ou vendas (histórico, status, detalhamento, etc)

    Gestão de estoque e catálogo

    Autenticação de usuários para acesso a informações sensíveis e/ou privadas

  • Pergunta 2

    • Para marketplaces confiáveis, a segurança dos dados torna-se imprecindível, visto que lidam com compra e venda, involvendo dados particulares e manuseio monetário.

    Navegação intuitiva e compatibilidade com diferentes plataformas (computadores, celulares, etc) são recomendados.

    Manutenibilidade para facilitar adicionar novas funcionalidades e/ou remover erros.
